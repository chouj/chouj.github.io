OPML 2.0 (Draft) 发布

@Chouj · Jul 19, 2007 · 1 min read

7月16号，Dave Winer 发布了OPML格式的2.0版草案，按原文的意思，该文档“未正式公布，未启用”，2.0版的OPML格式还未“最终确定”，此文档仅供讨论、参考。 目前我们所见到的OPML文件多为1.0或1.1的格式，具体可参见如下代码： <?xml version=“1.0” encoding=“UTF-8”?> <?xml-stylesheet type=“text/xsl” href=“/say/css/opml.xsl”?> OPML 2.0草案同样详细解释了如上各属性标签的含义和作用，原文中绿色标注的“ownerID”属性，是1.0版中所没有的一个新属性。 OPML 2.0何时正式发布，尚在未知阶段，相信时日不远。其发布将带来什么样的影响，各线上线下服务是否会依此调整以支持新格式，让我们拭目以待。先让我们补习一下： OPML是Outline Processor Markup Language的缩写，致力于成为一种简洁公开易读易写的文件格式，常被作为Feed阅读器和聚合器中订阅条目的文件载体。 [ 点击进入OPML 2.0 （Draft）] Powered by ScribeFire.

RSS聚合器 之 Gregarius

@Chouj · Jul 14, 2007 · 1 min read

什么是RSS聚合器？ RSS聚合器是一种以用户被动更新模式更新，可读取RSS Feed / XML并显示新闻项的程序，可以方便地在线获取、阅读和管理RSS/XML格式的信息。 Gregarius 个人认为Gregarius已经是一款相当成熟的RSS feed聚合器程序了，具备不错的用户体验，易于操作和管理。抽儿就是拿这款程序架设的RSS阅读器。特性如下： 支持 RDF, RSS, ATOM feeds 支持智能识别； 基于浏览器Web端的前台浏览和后台管理； 支持目录分类，支持标签系统，并运用了ajax技术； 支持搜索； 支持OPML； 可显示favicon； 支持已读未读分类和条目数量显示； 具有多种主题样式，支持插件； 前台样式： 后台含八个管理标签，分别为“控制面板”、“Feed管理”、“条目管理”、“设置”、“插件”、“主题”和“分类目录”，操作简单。 Demo站点： OpenRSS 话语权 Gregarius运行环境要求： Apache服务器（推荐） PHP (PHP 4.x 或 PHP 5.x, 4.3+ 推荐) 数据库 MySQL 和 SQLite Gregarius主页 | Gregarius 0.5.4 原版下载页面 | Gregarius 0.5.3 的安装说明 最后提一下抽儿在使用中的一些感受。首先，空间的访问速度和php的版本会直接影响feed添加以及OPML导入的成败，某些时候可能feed添加成功，但没有成功读取到内容，或抓取后不能实时更新，这在话语权中表现比较明显。其次，Gregarius有经过汉化的版本，搜索可得（比如这里），但汉化的不彻底，更新时间显示为乱码，可能涉及编码UTF-8/GB2312的支持和时间日期代码格式的问题。还有就是Gregarius对Feedsky的Feed不友好。 Powered by ScribeFire.

毕业了（三）

@Chouj · Jul 13, 2007 · 1 min read

纯记录，可无视。 前些天在饭否闹了个大笑话，把“杨远骋”喊成了“杨远聘”，不知道北航的这位大当家会怎么想。其实第一次看到他的名字就下意识的以为是“聘”，一时没改过来，思维定势的厉害。最近在他blog看到这么一个短语：Social的乐趣，说的很有感觉。毕业的若干天里，游行是Social的乐趣，摆摊卖书也是Social的乐趣。 接游行那一晚。自然上午睡足，下午即被吆喝着卖书，二话没说，拾掇一下就大麻袋小包得赶集去了。其实早就想把我那一堆东西处理处理，可惜有人来宿舍买书的时候，都是DotA最火热的时候，无法兼顾其他，只好憋到最后几天清仓。 书一撂，摊子一支，再美其名曰“一班专场”，罗敏同学更是抄出从西安华山带回来的拨浪鼓，一阵high敲，这阵仗算是摆开了。要说这拨浪鼓，被罗敏同学视为招财进宝的神器，引得二三十人问津也不卖；谁觉得自个儿的摊儿人气不旺，就都拿过来耍上一下，好生热闹。 话说卖书的Social乐趣，可谓花样迭出。所谓Social，不就是指人与人之间嘛。那卖书牵扯的 关系就不少：摊主与上帝的，摊主与路人的，摊主与学弟学妹熟人的，摊主与摊主的，摊主与收破烂儿的，摊主与委托人的，甚至摊主和体育爱好者的，各有风味。 为什么和体育爱好者呢？因为摊子在操场边儿上，因此引发了一场瓜葛，暂且按住不表。 在下课放学吃饭回寝的黄金时段之前，历来是要和捡便宜的大爷大妈们周旋下的。这些群众与学生的差别在 于，死压价不松口，我们在这淡季自然瞅准了这么个扯闲天儿，练口舌的机会，和大爷大妈们你来我往，代表性语句是：“保证坏不了！”其实我们也知道他们的生 活也不宽裕，正好我们的原则是尽量出货，所以便宜卖也不心疼，因为旗号是抚慰群众嘛，自个儿给自个儿升官加爵先。紧接着黄金时期一到，人多人气旺，忙活开 来。上帝们瞄什么，咱就跟着介绍什么，喊个凑合价，重要的是得享受光明正大和陌生人聊天的乐趣，施展下嘴皮子工夫，顺带算是经商演习。现在寻思下哈，大学 四年，除了我搞通讯的时候和陌生人联络联络，也就卖书这茬儿见的陌生人多了。 摊主和路人的段子，在于商业头脑和性格因素。咱形而下的说就是得有胆量向路人吆喝，有广告，有眼球效 应，咱才有门儿啊。不过乐不在此，您想人气旺的时候，吆喝是为生意，人气淡的时候，吆喝为什么呀？为美女呀。其实摆摊本就是个公然当街瞄美女的机会，有美 女路过，先叫唤下以吸引注意，能让美女留步赚得谈上两句的机会就更好了，别忘了，我们有拨浪鼓，那可是吸引MM的利器。 见着熟人，自是好事，不用不好意思。熟人的作用多啦：当托儿的来抬抬人气；想尝鲜的直接客串下 临时摊主，要么自个儿吆喝着玩儿，要么欣赏我们吆喝；碰见来扫荡的学弟学妹，自可吹嘘一番，这位考哪哪哪儿了，那教材怎么怎么滴；淡季的时候路过一朋友， 还可以解解闷儿。噢对，有点儿不好，举个例子：“哟，下海啦？收成咋样？发财了可得请我吃饭！” “你这儿人气不行啊！”，“哎，有零钱不？”，“今儿个卖了多少？”，摊主也摊主间的关系正是构架于人气指数、找钱能力、营业额这仨量度之上，按这整三个榜单，实时给出排行，咱就可以弄个“摆摊2.0”的帽子戴戴。当然彼此间还是合作无间的，以那传递着的拨浪鼓为证。 和委托人的关系，发生在卖书的首环节：总有些偷懒的，会委托我们卖点儿东西。和收废品的关系则发生在尾环节：实在出手不能，才会出此下策。委托人一般不关心价格，只关心是否售罄，没准儿可以从中剥削一道儿；收废品的更不关心价格，钉死了一块二一公斤，知识就是介末廉价。 最后说到和体育爱好者的关系，这档子事儿是个特例，算是毕业几天里特别疯狂的一笔，因为是场群架，直接惊动校保卫处和110，有兄弟受伤送往医院。不知道兄弟的伤好了没有，这也是毕业之后我挂念的事情之一。 透漏些有用的信息，毕业摆摊儿什么东西最好卖： 电器。包括电风扇、手电筒、电脑周边、充电器、电饭锅； 体育音乐器材。拳击套，球拍，球，口琴之类。 还有好卖的，就是别人没有你有的东西：少见的教材，特别外文原版那种；烟灰缸之类小玩意儿也特吃香。至于经验，可以总结出以下2条，仅供参考： 敢于推荐； 坚定的相信什么玩意儿都能卖出去。 多跟上帝介绍介绍你的货，多说有益。我们班上星仔连用剩的半壶食用色拉油都卖出去了，你说还什么卖不出去。 毕业之时，什么事情都该走上一遭，游行是，卖书也是。 另，摆弄feedburner时，不幸手抖，把http://feeds.feedburner.com/xuchi删到30天转向了，过程不可逆，不知道日后会如何。只好专一于Feedsky的feed，地址如下： http://feed.xuchi.name
