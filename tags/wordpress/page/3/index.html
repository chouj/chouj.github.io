[图文教程]给目录页或标签页添个Feed

@Chouj · Sep 22, 2007 · 1 min read

WordPress架构下的目录页和标签页不支持Feed输出。当我们想让自己的目录也支持Feed输出，或者，当我们只对blog某目录下的文章感兴趣的时候，想订阅这个目录，该怎么办呢？我们可以借助HTML2RSS工具来实现！ 以Feed43.com为例，抽儿PS了个教程， 由于图片硕大，就不贴在这里了，只放出LINK。依照教程思路，其他HTML2RSS工具同样能帮你给目录页或标签页添个Feed。 [ 点此进入图文教程 ] Update: 其实，在目录链接和标签链接后面加上/feed/就是目录页feed和标签页feed了。。。。我真傻，真的。 比如：http://aboutrss.cn/category/wordpress/feed/
