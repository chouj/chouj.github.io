MonitorThis:实现搜索结果的订阅

@Chouj · Jun 13, 2007 · 1 min read

每天网络上将产生许多新的页面，而搜索引擎的爬虫们在积极地抓取，努力把所有网络上的内容都纳入其搜索引擎中。如果能让爬虫们为你服务，将你关心的关键字的内容抓取到，再通过RSS推送到你面前，这是不是很激动人心呢？那么MonitorThis就可以帮到你的忙，她能让你订阅到你需要的关键字在22个搜索引擎上的结果。MonitorThis直译的意思是：“监视这个”，对于MonitorThis的功能来说，非常贴切。而22个搜索引擎分别为：

  • Google Blog Search
  • Google News search
  • Technorati tag search
  • del.icio.us tag search
  • furl search
  • Flickr tag search
  • MSN search
  • MSN News search
  • Yahoo search
  • Yahoo News search
  • Icerocket search
  • Feedster rss search
  • Topix.net search
  • Feedsfarm.com search
  • Newspad PRWeb search
  • Search4RSS rss search
  • Blogdigger search
  • Plazoo search
  • NewsNetPlus search
  • Blogmarks tag search
  • Find articles search
  • Wired News search
  • blogg.de tag search

OK，让我们开始，我们需要做的只有三步：

  1. 在MonitorThis页面上的“enter search term”中填入你所关心的关键词，比如“Google”之类（对中文支持不佳），点击右侧的“make monitor.opml”；
  2. “your monitor.opml”中即生成opml文件的代码，点击“select all”，拷贝到记事本中，保存为monitor.opml文件。高手可以直接更改opml文件的代码，一个“outline”标签即对应一个搜索引擎的关键词搜索，可自行添加或删减，再存储为opml文件；
  3. 一般的RSS阅读器都支持opml文件导入。将该文件导入阅读器后即出现22个订阅，可阅读可管理。最好管理下，将22个订阅归为一类，再删掉不需要的搜索引擎的结果订阅。

通过３步，爬虫们找到的新东西就可以实时出现在你面前了，很实用的功能，可以拿来监视某关键字在搜索引擎中的流行程度，监视相关关键字的新信息，等等。

PS.最近RSS界同样是新消息不断：抓虾改版，Feedsky改版，著名RSS搜索引擎Technorati解禁，FeedBurner将于6月中旬并入Google Account，纷纷杂杂，好不热闹。
