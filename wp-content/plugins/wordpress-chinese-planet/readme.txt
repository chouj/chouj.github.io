中文 Dashboard

== Description ==

把 incomming links 改成 Technorati，把 Development Blog 改成 WordPress 中文团队博客，把 Other WordPress News 改成 WordPress 火星

== Installation ==

1. 上传插件到 `/wp-content/plugins/` 目录下
1. 在 WordPress `Plugins` 菜单下激活该插件
